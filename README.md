# go-mining

Go-mining is a small library for data mining.

The library is written in [Go language](golang/go).

## Features

### Classifiers

* CART
* Random Forest
* Cascaded Random Forest

### Resampling

* SMOTE
* LN-SMOTE (Local Neigbourhood SMOTE)

### Miscellaneous

* Computing Gini index
* Computing K-Nearest Neighbourhood
